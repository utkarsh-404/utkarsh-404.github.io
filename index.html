SubEnum

A Bash script to automate subdomain enumeration, live target detection, and reconnaissance using popular tools like subfinder, sublist3r, knockpy, httprobe, and eyewitness.

GitHub

This tool offers a streamlined workflow for reconnaissance and subdomain enumeration. It uses 8+ tools to gather subdomains, providing comprehensive coverage across multiple sources. For live target detection, it filters active HTTP/HTTPS services using `httprobe`. The reconnaissance phase includes capturing screenshots of live targets with `eyewitness` and identifying technologies in use through `whatweb`. All of this is automated with a single command, making the process efficient and easy to use.
